About OptiCrop

Smart Agricultural Production Optimization Engine

Our Mission

OptiCrop was built to help farmers, students and agri-tech enthusiasts make faster, data-informed decisions about what to grow, how to fertilize it, and what yield to expect — combining Machine Learning with practical agronomic knowledge in one simple web app.

How It Works
  • Crop Recommendation uses a RandomForestClassifier trained on 2,200+ soil samples spanning 22 crops (N, P, K, temperature, humidity, pH and rainfall) to recommend the most suitable crop with a confidence score.
  • Fertilizer Recommendation compares your soil's NPK profile against published ideal ranges for the selected crop and suggests a corrective fertilizer action.
  • Yield Prediction is a demonstrative RandomForestRegressor estimate for education/prototyping purposes (see the README for details on this approximation).
  • Weather Integration pulls live conditions from the OpenWeather API so recommendations can be checked against real-time context.
